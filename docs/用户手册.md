# 用户手册

## 编辑题目

1. 从“文件”中新建经典或特色棋盘；特色棋盘输入逗号分隔的行轮廓，如 `4,5,4,5`。
2. 保持“编辑目标曲线”勾选，点击灰色边以添加或移除橙色目标曲线；滚轮缩放，右键或中键拖动画布。
3. 关闭“编辑目标曲线”后点击单元格，再用“设置数字”“设置 π”“清除提示”编辑题面。
4. “锁定/解锁”决定生成器能否替换该提示，锁定提示旁显示红点。
5. 先执行“校验曲线”，再运行“生成”。对话框可设置 10–1800 秒、π 自动/必须/禁用和初级、中级、高级、专家四种目标难度；推荐高级和专家题使用 300 秒以上。生成器会先保留一个已证明唯一的全提示基线，再持续删减提示，因此优化超时不会丢失已有可靠结果。

## 求解与状态

“求解”会区分不合规、无解、唯一解、多解和超时未判定。唯一解会写回题目文件；勾选“显示答案”可在画布中查看蓝色曲线。超时不等于多解或无解，状态保持未判定。精确求解默认最多使用 8 个 CPU 工作线程。

界面跟随系统深浅色主题，棋盘数字、π、目标曲线和答案线会自动选择高对比度颜色。

## 练习模式

1. 打开任意例题，点击工具栏或“练习”菜单中的“练习模式”。作者目标曲线和已存答案会立即隐藏，计时从 00:00 开始。
2. 点击棋盘边添加或删除自己的蓝色作答曲线。作答只保存在内存中，不会改动题目 JSON、目标曲线或答案。
3. 点击“验证作答”。系统会检查数字、π、顶点度数以及是否为单一闭环；错误时给出原因并继续计时，正确时停止计时并显示总用时。
4. “重置练习”会清空作答并重新计时。练习中重新加载题目也会自动清空上一份作答。
5. 退出练习模式不会保存玩家曲线，也不会显示答案。

## 文件与导出

题目以 UTF-8 JSON 保存。保存使用同目录临时文件原子替换。SVG 适合继续排版，PDF 为 A4 单题页；显示答案时导出的文件包含答案，否则只包含题面。纸笔题面中的候选边采用浅灰虚线，黑色钢笔或铅笔描出的答案会保持清晰。

## 活动题包

`output/puzzles` 保留历史目录名以兼容已有路径：`easy` 与 `medium` 现共同归为初级，`hard` 为中级，`level-4` 为高级，`level-5` 为专家。当前正式题包共 20 道，其中高级和专家各 4 道。`output/pdf` 提供完整题册和答案册：第 2 页为详细规则和两道递进式完整题解示例，之后每道正式题独立成页。活动不设置时间限制，也不在题面收集姓名、班级或用时信息。正式活动前，组织方仍可另行安排内部试玩以检查难度和规则理解。
